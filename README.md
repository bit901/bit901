## 🚀 Tech Stack
- **Frontend:** Next.js (React + Tailwind)
- **Mobile:** Expo (React Native)
- **Backend:** Node.js + Supabase
- **Integrations:** PostHog (Analytics), OneSignal (Push Notifications)
- **Deployment:** Vercel + Expo EAS
